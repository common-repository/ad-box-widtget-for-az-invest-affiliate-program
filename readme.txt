== Description ==

Widget designed to help new affiliates quickly get their affiliate link on WordPress based sites.
The plug-in requires a personal affiliate link, which can be obtained from the affiliate program's page ([http://www.az-invest.eu/affiliate-program](http://www.az-invest.eu/affiliate-program))

The widget can be placed on the sidebar or anywhere your currently used theme permits. This is done from the Appearance/Widgets menu. The featured affiliate link has a "*rel=nofollow*" attribute, which is in line with Google's guidelines and will not get you in trouble with google for having outgoing links on your blog or website. 

== Installation ==

1. Download the ZIP file containing the plug-in
2. Install the plug-in using from the Plugins/Add new menu using the Upload link.
3. Activate your plug-in
4. Make the plug-in visible from the Appearance/Widgets menu
